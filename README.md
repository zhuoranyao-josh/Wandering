# Wandering — AI 驱动旅行规划应用

🌍 演示页面（可点击访问 / 下载 APK）:  
https://wandering-landing-page.vercel.app

**Wandering** 是一个基于 Flutter 的 AI 旅行规划平台，将目的地探索、Checklist 行程收集和 AI 智能日程生成整合在一个应用中。用户可以在 3D 地球上浏览城市、添加目的地到旅行清单，并生成预算感知的每日行程。

---

## 功能

- **3D 地球探索**：使用 Mapbox SDK 浏览城市和景点，支持平滑缩放和自定义标记  
- **Checklist 系统**：将目的地、酒店、餐厅、活动和交通选项收集到结构化“旅行购物车”中  
- **AI 行程生成**：基于 Gemini API，根据用户清单、旅行日期、预算和偏好自动生成每日行程  
- **Firebase 集成**：用户认证、实时数据库存储（Firestore）、图片上传（Storage）、后台功能（Cloud Functions）  
- **地点信息丰富**：Google Places API 获取酒店、餐厅和活动的真实名称、地址、照片和地图链接  
- **天气信息**：OpenWeather API 提供旅行目的地的天气预报  
- **管理与国际化**：支持中英文 i18n 国际化，同时包含管理员模式进行内容管理  

---

## 技术栈

- **前端:** Flutter & Dart  
- **后端:** Firebase（Auth、Firestore、Storage、Cloud Functions）  
- **AI 行程规划:** Gemini API  
- **地图与地点:** Mapbox SDK、Google Places API  
- **天气服务:** OpenWeather API  
- **路由管理:** GoRouter  
- **开发环境:** VS Code, Android 真机测试
